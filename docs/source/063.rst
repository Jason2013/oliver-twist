第三十一章(3)
================

简而言之，经过若干进一步的调查，费了许多口舌，治安推事才欣然同意梅莱太太和罗斯伯力先生联名保释奥立弗，但必须随传随到。布拉瑟斯和达福拿到两畿 尼的酬金，回伦敦去了，但他们二位对这次远行的目的却有不同的见解。后一位绅士纵观全局，考虑再三，倾向于相信这一次未遂夜间行窃系高手佩特所为。而前一 位在同等程度上倾向于把这一功绩整个算在了不起的大烟囱契科韦德先生头上。

此时，在梅莱太太、露丝和心地善良的罗斯伯力先生齐心照料下，奥立弗的身体日趋康复。如果说发自内心，洋溢着感恩之情的热切祈祷能够上达天听——否则还成其为什么祈祷——那么，这个孤儿为他们祈求的祝福已化作宁静与欢乐，渗入了他们的心灵。
